# Vigilante das Rochas do Mar

## Summary
Este projeto utiliza inteligência artificial para monitorizar e identificar alterações nas rochas marinhas, ajudando na preservação dos ecossistemas costeiros através de análise de imagem.

## Background
A erosão costeira e o impacto humano nas rochas marinhas são problemas crescentes. Este projeto visa automatizar a vigilância para detetar riscos biológicos ou geológicos precocemente.

## How is it used?
O sistema pode ser usado por biólogos marinhos e autoridades costeiras através de câmaras fixas ou drones que enviam imagens para processamento.
